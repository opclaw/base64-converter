# Base64 Converter

Free online tool to encode and decode Base64 strings and files.

## Features

- ✅ Text to Base64 encoding
- ✅ Base64 to text decoding
- ✅ File to Base64 encoding
- ✅ Copy & download results
- ✅ 100% client-side (private & secure)
- ✅ Mobile responsive

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# Output in /dist folder
```
